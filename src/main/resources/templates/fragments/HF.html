About Project

PoliSight: 청년 정책 시뮬레이터

복잡한 정책 공고문을 AI가 분석하여, 사용자의 상황에 맞는 수혜 가능성과 예상 지원금을 시뮬레이션해주는 서비스입니다.

주요 기능

  • RAG(검색 증강 생성) 기반 정책 공고 분석
  • 개인별 조건(소득, 자산 등)에 따른 맞춤형 시뮬레이션
  • AI 정책 안내원과의 대화형 상담

Tech Stack

Spring Boot MyBatis LLM RAG
